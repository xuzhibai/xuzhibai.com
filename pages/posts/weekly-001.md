## 开篇：为什么开始写周刊

这周终于下定决心，开始写这份周刊。

其实想搭建自己的博客很久了。一个是懒，另一个是觉得没什么可写的内容。后来受到[猫鱼周刊](https://ameow.xyz/)的启发——博主每周分享自己读过的内容，想到我自己平时也会像这样收藏一些有意思的内容，于是也决定做个"周刊博主"，分享出来。

我也想过，我认为值得一看的内容，是否真的值得一看。

我们每个人都被困在自己的"信息茧房"里。我们现在所说的专业，岗位，对一个群体来说是常识的东西，对另一个专业的人可能是完全陌生的领域，但是生活中我们又需要接触到各个领域。记得上大学那会，有个朋友去电脑店装系统被收了200块。而对我来说，重装系统不过是顺手的事。

有句话是说：很多赚钱的方法，赚的就是信息差。似乎确实是这样。

这也是我想做这份周刊的原因——从一个程序员的视角，分享一些我认为值得看的内容，以及适合普通人上手的工具、开源项目。

希望对你有用。

## 值得一看

### 1. 疲惫经济学：为什么技术进步了，我们却更累了？

[原文链接](https://youzhiyouxing.cn/materials/2148)

这是一期播客，讨论了一个反直觉的现象：技术进步并没有让我们更轻松，反而可能更累。

一个让我印象很深的观点是——很多生意赚的其实是人们"疲惫的钱"。外卖、效率工具、便利服务，表面上是帮你省时间，实际上是在利用你的疲惫创造需求。更深层的一个洞察是：劳动强度与生产工具无关，与生产关系有关。8小时工作制不是技术进步带来的，而是工人争取来的，也分析了为什么996如此普遍。

还有一个让人细思极恐的现象：未来AI 可能不只是工具，还会成为监工。现在就有来监控员工行为的软件，AI 时代可能更进一步——绩效分析、工作量化、自动提醒老板你有没有在摸鱼。

### 2. 从零开始两天构建一个 Claude Code

[原文链接](https://x.com/IceBearMiner/status/2037888800341610684)

作者用两天时间从零构建了一个 Claude Code，并详细记录了整个过程。分析了一个生产级的 agentic CLI 都需要哪些组件：System Prompt 的分段架构（为什么把工具定义和输出格式分开）、Prompt Caching 的实现（如何节省 token）、Agent Loop 的状态机设计（工具调用 → 执行 → 结果返回的循环）、SSE 缓冲管理（流式输出的技术细节）。

如果想要开发AI agent应用，可以看看。

### 3. 未来最好的人生策略，是跳出单一收入陷阱

[原文链接](https://m.okjike.com/originalPosts/69b8c47c4d08cc0d63061982)

这篇文章用数据阐述了一个观点：我们过于依赖工资收入了，而工作的工资增长“投产比”越来越低，可能花一倍的时间去职场内卷，也换不来一年10%的涨薪，所以我们应该让收入多元化。

一直以来对此也是深有体会，不过我的想法是尽快实现“财务自由”。

之前读过一本书叫《**富爸爸穷爸爸**》，其中介绍了一个叫"老鼠赛跑"的现象，指的是大多数人的一生重复着这样一种循环：努力上班赚钱，收入增加，但是生活开支也随之增加（买更好的车、更大的房子、更贵的消费，于是背上了贷款和负债）。为了维持这种生活方式，就会一直重复“工作-赚钱-消费-负债”的循环。

只有当被动收入能够覆盖甚至超过日常总支出时，才能跳出这个循环，也就实现了”财务自由”。感兴趣的朋友可以看看这本书。

### 4.你的下一个员工，何必是同事 - 女娲 Skill

[原文链接](https://m.okjike.com/originalPosts/69d1ea9d5effd6c397382949)

最近网上把离职的同事变成skills装进AI里很火，更离谱的还有前任skills，只要提供聊天记录就能把AI变成前任。这个作者的项目更进一步，写了个女娲skill，可以把任何人都变成skills，先不说蒸馏后的skills是否真的有效果，流浪地球里说的数字生命没想到这么快就实现了，有点赛博朋克的感觉了。

## 哆啦百宝箱

### 1. Port Killer — 端口管理工具

[项目链接](https://github.com/productdevbook/port-killer)

一个跨平台的端口管理工具，能自动发现所有监听的 TCP 端口，一键终止进程。支持优雅终止和强制终止，还能按端口号或进程名搜索筛选，在macOS中是个菜单栏应用，美观好用，也支持Windows。

非常适合我这种隔三差五就忘记终端命令的开发者。

### 2. ZTools — uTools 开源替代

[项目链接](https://github.com/ZToolsCenter/ZTools)

应用启动器 + 插件工具集，一个uTools的开源替代。

uTools可以说是我的装机必备了，启动应用、搜索只需要按两下command（macOS），类似于macOS的 Spotlight，不过uTools最好用的还是它的插件生态，包括各种办公效率工具，比如我常用的json解析、截图工具、图片识别复制翻译。

现在uTools 免费版限制越来越多，ZTools的出现也是个好消息。目前插件还不多，但基础功能已经可用，期待后续发展。

### 3. MacCalendar — 菜单栏日历

[项目链接](https://github.com/bylinxx/MacCalendar)

开源的菜单栏日历，支持中国农历、节假日、放假安排，还能同步系统日程。完全免费，离线使用，小而美。不过要注意仅支持 macOS。
